# 多节点配电网样例文档

## 概述

本样例 `multi_node_feeder.json` 是一个包含30个节点的中等规模径向配电网拓扑，专门用于测试Julia-Grid Designer & Optimizer的各项功能。

## 网络特点

### 基本参数
- **节点总数**: 30个母线节点
- **负荷节点**: 8个负荷元件
- **电源**: 1个主电源 + 4个分布式电源
- **线路**: 24条配电线路
- **开关**: 5个开关（支持网络重构）
- **基准容量**: 100 MVA
- **基准电压**: 10.5 kV

### 拓扑特性

1. **径向结构**: 典型的配电网径向结构，从主电源向外辐射
2. **多分支**: 分为3个主要分支（A、B、C），每个分支又有多个子分支
3. **分布式电源**: 包含光伏、风机、微型燃机、储能等多种类型的DG
4. **开关重构**: 5个开关可用于网络重构测试
   - `sw-1`: 连接A分支末端和C分支末端
   - `sw-2`: 连接B分支和C分支
   - `sw-3, sw-4`: 正常闭合的联络开关
   - `sw-5`: C分支的备用连接

## 预期计算结果

### AC潮流分析
- **平衡节点**: bus-1 (主电源)
- **PV节点**: dg-1, dg-2, dg-3 (分布式电源)
- **PQ节点**: 所有负荷节点
- **预期收敛**: 5-8次迭代
- **电压范围**: 0.95-1.05 p.u.

### 拓扑重构优化
通过开关状态调整，可以实现：
1. **负荷均衡**: 重新分配各分支负荷
2. **网损降低**: 优化潮流分布，减少线路损耗
3. **电压改善**: 提升末端节点电压水平
4. **DG消纳**: 提高分布式电源的利用率

## 注意事项

1. **平衡节点**: 确保bus-1的is_slack标记为true
2. **DG状态**: 所有分布式电源的status应设为1（投入运行）
3. **开关状态**: 初始状态有些开关为OPEN，可用于测试重构效果
4. **参数合理性**: 所有参数已优化，确保潮流计算能收敛
