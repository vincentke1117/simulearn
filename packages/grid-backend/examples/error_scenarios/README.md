# 错误场景测试样例

## 概述

本目录包含5个专门用于测试Julia-Grid Designer & Optimizer错误处理机制的错误场景样例。每个文件都包含一种特定的错误，用于验证系统能否正确识别、报告和处理各种错误情况。

## 测试场景列表

### 1. 孤岛网络 (isolated_network.json)

**错误类型**: 拓扑结构错误
**描述**: 网络中包含未与主电源连接的节点，形成孤岛
**错误特征**:
- bus-3和bus-4组成一个孤岛子网络
- 孤岛中有负荷但无电源
- 孤岛与主网之间无连接线路

**预期错误**:
```
ERROR: 孤岛网络检测到：节点 bus-3, bus-4 未连接到平衡节点
```

**测试目的**:
- 验证拓扑连通性检查
- 确认错误提示清晰易懂
- 测试系统不会崩溃

### 2. 缺少平衡节点 (no_slack_bus.json)

**错误类型**: 平衡节点缺失
**描述**: 网络中没有任何节点标记为平衡节点 (is_slack=true)
**错误特征**:
- 所有bus节点都未设置is_slack
- 存在分布式电源但未设置为PV节点
- 整个网络无电压参考点

**预期错误**:
```
ERROR: 未找到平衡节点，请设置至少一个节点的 is_slack=true
```

**测试目的**:
- 验证平衡节点检查
- 确保潮流计算有参考点
- 检查错误提示准确性

### 3. 阻抗参数错误 (invalid_impedance.json)

**错误类型**: 线路参数错误
**描述**: 线路阻抗参数不符合物理规律
**错误特征**:
- line-neg-r: 负电阻 (-0.10 Ω)
- line-zero-x: 零电抗 (0.0 Ω)
- line-large-r: 过大阻抗 (100Ω, 200Ω)

**预期错误**:
```
WARNING/ERROR: 线路 line-neg-r 电阻值为负数 (-0.10 Ω)
WARNING/ERROR: 线路 line-zero-x 电抗值异常 (0.0 Ω)
ERROR: 线路 line-large-r 阻抗值过大 (r=100.0 Ω, x=200.0 Ω)
```

**测试目的**:
- 验证参数范围检查
- 测试物理合理性验证
- 确认错误分级处理

### 4. 负荷参数错误 (invalid_load.json)

**错误类型**: 负荷参数错误
**描述**: 负荷参数不符合实际情况
**错误特征**:
- load-2: 负有功负荷 (-800 kW)
- load-3: 过大负荷 (50 MW)

**预期错误**:
```
ERROR: 负荷 load-2 有功功率为负值 (-800 kW)
ERROR: 负荷 load-3 功率过大 (p=50000 kW > 10000 kW 限制)
```

**测试目的**:
- 验证负荷参数范围
- 检查业务逻辑限制
- 测试参数合理性

### 5. 电压参数错误 (invalid_voltage.json)

**错误类型**: 电压参数错误
**描述**: 节点电压参数异常
**错误特征**:
- bus-1: 零电压 (0.0 kV)
- bus-2: 负电压 (-10.5 kV)
- bus-3: 过高电压 (1000.0 kV)
- bus-4: 异常vm_pu (5.0 p.u.)

**预期错误**:
```
ERROR: 节点 bus-1 电压值为零 (0.0 kV)
ERROR: 节点 bus-2 电压值为负数 (-10.5 kV)
ERROR: 节点 bus-3 电压值过大 (1000.0 kV)
WARNING: 节点 bus-4 电压幅值异常 (vm_pu=5.0)
```

**测试目的**:
- 验证电压参数检查
- 测试电压范围限制
- 检查多级错误处理

## 错误处理机制验证

### 前端错误处理

1. **错误捕获**
   - [ ] 前端能捕获Julia返回的错误
   - [ ] 错误信息正确显示给用户
   - [ ] 系统不发生崩溃

2. **错误提示**
   - [ ] 错误信息清晰易懂
   - [ ] 错误位置明确指出
   - [ ] 提供解决建议

3. **用户交互**
   - [ ] 错误后可以继续操作
   - [ ] 支持错误后重新导入
   - [ ] 有撤销/重做机制

### 后端错误处理

1. **参数验证**
   - [ ] 拓扑连通性检查
   - [ ] 参数范围验证
   - [ ] 物理合理性检查

2. **错误分类**
   - [ ] ValidationError: 参数验证错误
   - [ ] TopologyError: 拓扑结构错误
   - [ ] SolverError: 求解器错误

3. **错误报告**
   - [ ] 错误类型明确
   - [ ] 错误位置准确
   - [ ] 错误原因详细

## 错误分级

### 严重错误 (ERROR)
- 缺少平衡节点
- 孤岛网络
- 阻抗参数严重错误
- 负荷功率过大

### 警告 (WARNING)
- 参数超出正常范围但可接受
- 建议优化参数值
- 可能影响计算精度

### 提示 (INFO)
- 参数接近限制
- 建议检查参数
- 计算可能收敛较慢
