# 标准样例说明文档

## 概述

`sample_topology.json` 是Julia-Grid Designer & Optimizer项目的标准配电网样例，专门用于基础功能测试和演示。该样例包含最简单但完整的配电网结构，验证系统的核心功能。

## 网络结构

### 基本参数
- **节点数**: 2个母线节点
- **负荷数**: 1个负荷
- **电源**: 1个主电源 + 1个分布式电源
- **线路**: 1条配电线路 + 1个开关
- **基准容量**: 100 MVA
- **基准电压**: 10.5 kV

### 拓扑结构

```
bus-1 (平衡节点) ──── line-1 ──── bus-2 (负荷节点)
    │                                 │
[grid-1]                           [dg-1]
主电源                          分布式电源
  │                                │
[load-1]                         (无功补偿)
配网负荷
```

## 元件参数

### 母线节点

| 节点ID | 类型 | 名称 | 电压 (kV) | 平衡节点 | 电压幅值 | 相角 |
|--------|------|------|----------|----------|----------|------|
| bus-1 | Bus | 平衡节点 | 10.5 | ✓ | 1.0 p.u. | 0.0° |
| bus-2 | Bus | 负荷节点 | 10.5 | - | - | - |

### 电源设备

#### 主电源 (grid-1)
- **位置**: bus-1
- **类型**: 同步发电机
- **有功**: 1000 kW (额定)
- **有功范围**: 0 - 2000 kW
- **无功**: 300 kVar (额定)
- **无功范围**: -800 - 800 kVar
- **状态**: 投入运行

#### 分布式电源 (dg-1)
- **位置**: bus-2
- **类型**: 光伏/小型电源
- **有功**: 200 kW (额定)
- **有功范围**: 0 - 300 kW
- **无功**: 50 kVar (额定)
- **无功范围**: -150 - 150 kVar
- **状态**: 投入运行

### 负荷设备

| 负荷ID | 名称 | 连接母线 | 有功 (kW) | 无功 (kVar) |
|--------|------|----------|----------|------------|
| load-1 | 配网负荷 | bus-2 | 800 | 200 |

**总负荷**: 800 kW + 200 kVar

### 线路设备

#### 配电线路 (line-1)
- **起点**: bus-1
- **终点**: bus-2
- **电阻**: 0.10 Ω
- **电抗**: 0.30 Ω
- **容量**: 10 MVA

#### 开关 (sw-1)
- **起点**: bus-1
- **终点**: bus-2
- **状态**: CLOSED (闭合)
- **电阻**: 0.05 Ω
- **电抗**: 0.15 Ω

## 功率平衡

### 发电侧
- **主电源**: 1000 kW
- **分布式电源**: 200 kW
- **总发电**: 1200 kW

### 负荷侧
- **总有功负荷**: 800 kW
- **总无功负荷**: 200 kVar
- **线路损耗**: 约 20 kW (估算)
- **开关损耗**: 约 5 kW (估算)

### 平衡关系
- **发电 = 负荷 + 损耗**
- 1200 kW = 800 kW + 20 kW + 5 kW + ... ✓
- **基本平衡，符合物理规律**

## 计算预期

### AC潮流分析

**节点类型**:
- bus-1: 平衡节点 (Slack Bus)
  - 电压: 1.0∠0° p.u.
  - 功率: P = 1000 kW, Q = 300 kVar (调节)

- bus-2: PQ节点
  - 电压: 约 0.97∠-2° p.u. (预期)
  - 负荷: P = 800 kW, Q = 200 kVar

**收敛指标**:
- 电压偏差: < 1e-6
- 功率偏差: < 1e-6
- 迭代次数: 3-5次
- 计算时间: < 0.1秒

### 预期结果

| 指标 | bus-1 | bus-2 | 线路 |
|------|-------|-------|------|
| 电压幅值 (p.u.) | 1.000 | 0.970 | - |
| 电压相角 (°) | 0.0 | -2.0 | - |
| 有功功率 (kW) | 1000 | -800 | 190 |
| 无功功率 (kVar) | 300 | -200 | 95 |
| 线路损耗 (kW) | - | - | 20 |
| 负载率 (%) | - | - | 19 |

## 使用方法

### 1. 前端导入
```bash
$ 打开前端画布
$ 点击"导入JSON"按钮
$ 选择 examples/sample_topology.json
$ 画布将显示2个节点和1条连线
```

### 2. 运行潮流
```bash
$ 点击"运行潮流"按钮
$ 等待计算完成 (约0.1秒)
$ 查看右侧结果面板
```

### 3. 查看结果
```json
{
  "status": "ok",
  "type": "ac_pf",
  "buses": [
    { "id": "bus-1", "vm_pu": 1.000, "va_deg": 0.0 },
    { "id": "bus-2", "vm_pu": 0.970, "va_deg": -2.0 }
  ],
  "branches": [
    { "id": "line-1", "p_mw": 0.190, "q_mvar": 0.095, "loading_pct": 19.0 }
  ],
  "summary": {
    "loss_mw": 0.020,
    "iter": 4
  }
}
```

### 4. 运行优化
```bash
$ 点击"拓扑重构+DG优化"按钮
$ 观察开关状态变化
$ 查看优化结果
```
