1\. 项目概述

1.1 产品定位

一个面向中国市场的Simulink学习与AI辅助建模平台，结合交互式教育和智能建模助手两大核心功能。

1.2 核心价值主张

交互式深度教育: 通过项目沙盒、AI导师、调试训练器等功能提供沉浸式学习体验

智能AI建模助手: 替代中国市场无法使用的Simulink Copilot，提供自然语言建模能力

本土化深度优化: 针对中国工程师和学生的特定需求定制

1.3 目标用户

工科在校大学生 (自动化、电气、机械等专业)

初级在职工程师 (1-3年经验)

高校教师 (用于教学管理)

1.4 产品愿景与阶段目标（来自PRD）

愿景: 成为中国Simulink开发者与学习者的首选平台，通过交互式深度教育与强大的AI辅助，重塑学习与应用方式。

阶段目标:

- 近期（第一阶段，0-9个月）: 推出MVP，验证“教育+AI”核心价值；与≥3所重点高校合作试点；≥200名高质量种子用户；次月留存≥40%。
- 中期（第二阶段，9-24个月）: 用户规模化增长与商业模式验证；推广至≥50所高校；个人付费用户≥10,000；AI建模助手核心功能全面上线。
- 长期（第三阶段，24-36个月）: 构建生态与市场领导；打造活跃开发者社区，模型分享与协作驱动内容自增长；拓展企业级服务。

1.5 商业模式摘要（来自PRD）

采用免费增值与多层订阅：

- 免费版: 访问部分项目沙盒、基础问答（限额）、社区浏览。
- 个人专业版: 全部教育内容、完整AI建模助手、社区高级功能；订阅示例 ¥30/月 或 ¥298/年。
- 高校教育版: 含教师端（班级管理、自动批阅等），按席位数/学期/年打包报价。
- 企业版: 高校版全部功能 + 智能模型重构优化、团队知识库集成、可选私有化部署、定制培训；B2B定制化定价。



2\. 技术栈要求

主框架: React 18+ 或 Vue 3+ (优先React)

12. 非功能性需求（来自PRD）

性能

- 云端仿真响应：中小型模型仿真结果返回时间 ≤ 10 秒。
- AI助手响应：自然语言建模与问答平均响应时间 ≤ 5 秒。
- 平台并发：支持 ≥ 10,000 名用户同时在线进行常规操作。

可靠性

- 平台可用性（Uptime）≥ 99.9%。
- AI生成模型经闭环验证后，首次仿真成功率 ≥ 95%。

安全性

- 遵守《个人信息保护法》，用户数据（含模型与代码）加密存储。
- 提供企业级私有化部署选项，满足数据安全与合规需求。

易用性

- 界面简洁直观，新用户无需培训即可完成核心学习与建模任务。
- AI交互流程自然流畅，符合中文用户语言习惯。











7\. 核心功能模块详细说明

7.1 用户认证与授权模块

功能需求:

用户注册 (邮箱/手机号)

登录 (支持记住我)

第三方登录 (微信、GitHub)

密码重置 (邮箱验证)

角色权限管理 (RBAC)

页面流程:

注册流程:

&nbsp; 填写信息 → 邮箱验证 → 完善画像问卷 → 进入主控制台





登录流程:

&nbsp; 输入凭证 → 验证 → 跳转至上次访问页面或主控制台





权限验证:

&nbsp; - 路由级守卫 (某些页面需登录)

&nbsp; - 功能级权限 (如教师门户仅教师可见)

&nbsp; - API请求携带JWT Token



数据流:

用户操作 → 前端表单验证 

&nbsp;        → API请求 (POST /api/auth/login)

&nbsp;        → 后端验证 → 返回JWT Token
